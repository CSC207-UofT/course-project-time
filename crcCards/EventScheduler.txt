Class: EventScheduler

Responsibilities:
    - Methods that add/remove events that the calendar will contain
    - Convert all uncompleted tasks in a TodoList into a list of events, using the TaskToEvent converter
    - Create a calendar using the added events
    - Find an available time for a task to be scheduled as an event
    - Check if a time is available to schedule an event

Collaborators:
    - Calendar
    - Task
    - TodoList
    - TaskToEvent
    - Event
